# ficklemaster let's dance
## 任务目标 —— 视频中人脸表情检测
## 任务流程：
### 1.人脸识别
实现从视频中快速将人脸部进行识别，包括正脸和侧脸；
优化识别算法，提高识别准确率和效率；
### 2.表情识别
利用公开或者爬取人脸数据集进行面部表情识别，实现多分类任务；
优化分类算法，提高分类准确率和效率；
### 3.整合并优化模型
尝试将模型优化至工业级算法，实现实时分析和低配置部署。
## 技术手段
1. TensorFlow手工搭建模型
2. OpenCV类库调用及搭建
